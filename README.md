# Rugenraj — Angry Birds Slingshot Portfolio

An interactive portfolio built as a single HTML file with an Angry Birds-inspired slingshot game. Pull back the bird and launch it at targets to explore different sections.

## Features

- **Slingshot mechanics** — drag and release to launch the bird
- **3 target sections** — About Me, Skills, Projects
- **Moon easter egg** — hit the moon or launch backward to discover a hidden panel
- **Fully responsive** — works on desktop and mobile
- **Zero dependencies** — vanilla JS, single HTML file, no build step
